# snowyseoul

### A small package for ChosunPop  

+ 띄어쓰기 없는 역사자료 텍스트의 편리한 분해를 위한 패키지 (slice_by_idx, slice_by_str) <br/><br/>

+ 한자로 표기된 숫자의 아라비안 십진법 전환 (Chi2Fig) <br/>
+ 간지(干支) 산출을 위한 메소드 (ganji, ganji_gap, years_of_ganji) <br/>

<br/>

### 사용  
> pip install git+https://github.com/almostseoul/snowyseoul.git  
> from chosun_pop import *
